Cipher & Encoding Techniques Explained

Understand how each cipher works with easy-to-read descriptions and examples.

Caesar Cipher

A substitution cipher where each letter is shifted by a fixed number of positions.

Example: "HELLO" with shift 3 → "KHOOR"

Vigenère Cipher

Uses a keyword to shift letters using a repeating sequence of Caesar ciphers.

Example: Text: "ATTACK", Key: "LEMON" → Ciphertext: "LXFOPV"

Atbash Cipher

A monoalphabetic cipher where the alphabet is reversed. A ↔ Z, B ↔ Y, etc.

Example: "HELLO" → "SVOOL"

ROT13 Cipher

A special case of Caesar Cipher with a fixed shift of 13.

Example: "HELLO" → "URYYB"

Rail Fence Cipher

A transposition cipher where text is written in a zigzag pattern on multiple levels (rails).

Plaintext: "HELLO WORLD" with 3 rails → "HOLELWRDLO"

Columnar Transposition

Text is written in rows under column headers defined by the keyword and read column by column.

Plaintext: "WE ARE DISCOVERED" Key: "ZEBRAS" → Ciphertext: "EVLNEACDTESEAROF"

Playfair Cipher

A digraph substitution cipher using a 5x5 matrix of letters based on a keyword.

Example: "HELLO" + Key: "MONARCHY" → "GCNVNY"

Base64 Encoding

Binary-to-text encoding scheme representing binary data in an ASCII string format.

Example: "HELLO" → "SEVMTE8="

URL Encoding

Encodes special characters in URLs using % followed by two hex digits.

Example: "hello world" → "hello%20world"
